# Sky Markdown

一个 **Typora 风格** 的所见即所得编辑器。

核心特性

  • 实时渲染，无需分屏预览
  • 多标签页，像浏览器一样切换
  • 支持 行内代码、表格、任务列表

为写作而生的功能

抽取 Typora 的核心体验，去掉多余的负担。

✍️

真·所见即所得

基于 Milkdown / ProseMirror，输入 Markdown 语法即时渲染为成品样式，所写即所得。

🗂️

多标签页

每个文档一个标签，像文本编辑器一样在同一窗口里自由切换，未保存有小圆点提示。

🧭

大纲 / 文件侧边栏

自动生成标题大纲，一键跳转；内置文件树，打开文件夹直接浏览。

🔍

查找和替换

Ctrl+H 浮层式查找替换，支持区分大小写、逐个替换与全部替换。

🎨

多主题切换

内置 GitHub、Newsprint、Night、Pixyll、Whitey 五套主题，明暗随心。

📤

导出与源码模式

一键导出 HTML / PDF，随时切换源代码模式查看原始 Markdown。

🎯

专注 / 打字机模式

专注模式淡化非当前段落，打字机模式保持光标行居中，沉浸书写。

🔤

拼写检查与字数统计

状态栏一键开关拼写检查，实时字数、字符、行数与预计阅读时长。

⌨️

完整菜单与快捷键

文件 / 编辑 / 段落 / 格式 / 视图 / 主题全套菜单，配合熟悉的快捷键。

技术栈

现代、轻量、纯前端渲染

渲染层完全运行在浏览器内核中，跨平台能力天然具备；当前打包面向 Windows 发行。

Electron 33 TypeScript Milkdown v7 ProseMirror electron-vite electron-builder

三步开始使用

克隆仓库即可在本地运行开发版。

克隆仓库

git clone https://github.com/sky-carry/sky-markdown

安装依赖

进入目录后执行 npm install，等待依赖装好。

启动 / 打包

npm run dev 启动开发版，npm run build:win 生成 Windows 安装包。

开始你的纯粹写作

免费、开源，自己掌控的 Markdown 编辑器。
